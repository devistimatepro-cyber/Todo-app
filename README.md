# ✅ Todo List App

একটি আধুনিক এবং ব্যবহারকারী-বান্ধব-Todo List অ্যাপ্লিকেশন যা React.js দিয়ে তৈরি এবং Local Storage এ ডেটা সংরক্ষণ করে।

## ✨ বৈশিষ্ট্যসমূহ

✅ **কাজ যোগ করা** - নতুন কাজ সহজেই যোগ করুন  
✅ **অগ্রাধিকার সিস্টেম** - তিনটি স্তরের অগ্রাধিকার (কম, মাঝারি, উচ্চ)  
✅ **কাজ সম্পন্ন করা** - চেকবক্স দিয়ে কাজ চিহ্নিত করুন  
✅ **কাজ এডিট করা** - যেকোনো কাজ পুনরায় সম্পাদনা করুন  
✅ **কাজ ডিলিট করা** - অপ্রয়োজনীয় কাজ মুছে ফেলুন  
✅ **ফিল্টার সিস্টেম** - সব, সক্রিয়, এবং সম্পন্ন কাজ ফিল্টার করুন  
✅ **Local Storage** - সমস্ত ডেটা স্থায়ীভাবে সংরক্ষণ হয়  
✅ **ডার্ক মোড** - লাইট এবং ডার্ক থিম সুইচ করুন  
✅ **পরিসংখ্যান** - কাজের অগ্রগতি ট্র্যাক করুন  
✅ **রেসপন্সিভ ডিজাইন** - সব ডিভাইসে নিখুঁত কাজ করে  
✅ **বাংলা ভাষা সমর্থন** - সম্পূর্ণ বাংলায় ইন্টারফেস  

## 🚀 দ্রুত শুরু

### প্রয়োজনীয়তা
- Node.js (v14 বা তার বেশি)
- npm বা yarn

### ইনস্টলেশন

```bash
# রিপোজিটরি ক্লোন করুন
git clone https://github.com/devistimatepro-cyber/Todo-app.git
cd Todo-app

# নির্ভরতা ইনস্টল করুন
npm install

# ডেভেলপমেন্ট সার্ভার চালু করুন
npm start
```

ব্রাউজারটি স্বয়ংক্রিয়ভাবে [http://localhost:3000](http://localhost:3000) এ খুলবে।

### `npm test`

## 🎯 কীভাবে ব্যবহার করবেন

1. **নতুন কাজ যোগ করুন**
   - ইনপুট ফিল্ডে আপনার কাজ লিখুন
   - অগ্রাধিকার নির্বাচন করুন
   - "যোগ করুন" বাটনে ক্লিক করুন

2. **কাজ চিহ্নিত করুন**
   - চেকবক্সে ক্লিক করে কাজ সম্পন্ন করুন

3. **কাজ এডিট করুন**
   - ✏️ বাটনে ক্লিক করে কাজ পরিবর্তন করুন

4. **কাজ ডিলিট করুন**
   - 🗑️ বাটনে ক্লিক করে কাজ মুছুন

5. **ফিল্টার করুন**
   - বিভিন্ন ফিল্টার দিয়ে কাজ দেখুন

6. **থিম পরিবর্তন করুন**
   - 🌙/☀️ বাটনে ক্লিক করে থিম পরিবর্তন করুন

## 💾 ডেটা সংরক্ষণ

সমস্ত ডেটা ব্রাউজারের Local Storage এ সংরক্ষিত হয়। এর অর্থ:

- আপনার সমস্ত কাজ স্বয়ংক্রিয়ভাবে সংরক্ষিত হয়
- ব্রাউজার বন্ধ করলেও ডেটা হারায় না
- একাধিক ট্যাবে সিংক হয় না (স্বাধীন সংরক্ষণ)

## 🌙 ডার্ক মোড

হেডারের উপরে থিম টগল বাটন 🌙/☀️ ব্যবহার করে লাইট এবং ডার্ক মোড সুইচ করুন। আপনার পছন্দ Local Storage এ সংরক্ষিত হয়।
